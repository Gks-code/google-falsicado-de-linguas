About Language Explorer

Google is bringing the world's 7,000+ languages online to help make information more accessible for everyone. Language Explorer is an interactive visualization of the world's linguistic diversity, powered by LinguaMeta.

LinguaMeta is a unified resource for language data developed by Google Research. It provides metadata on language distribution, speaker populations, and vitality status.

Research Paper

This research paper details the introduction of LinguaMeta, a unified resource for language data developed by Google Research. LinguaMeta is an open-source repository of language metadata, including language codes, number of speakers, writing systems, official endangerment status, and more.

FAQ

What is Google Language Explorer?

It's a platform for exploring linguistic diversity through an immersive geographical interface, highlighting efforts to bring more languages online.

What is LinguaMeta?

LinguaMeta is the underlying dataset providing metadata for global languages, developed specifically for research and accessibility.

How should I cite this data?

Please refer to the 2024 LREC-COLING paper for official citation guidelines.
